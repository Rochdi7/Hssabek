────────────────────────────────────────
  N) TEMPLATES MARKETPLACE (SUPER ADMIN CONTROLLED)
────────────────────────────────────────

  BUSINESS MODEL:
  ───────────────
  • YOU (Super Admin) create & own all templates
  • Templates are PRODUCTS you sell to tenant companies
  • Each template has a unique ID — Super Admin assigns/revokes per company
  • Free templates come with every plan (basic set)
  • Premium templates require purchase OR Super Admin manual activation
  • Super Admin can grant, revoke, extend, expire any template for any tenant

  FLOW:
  ─────
  1. Super Admin creates template in template_catalog (global)
  2. Template is visible to ALL tenants (preview + locked state)
  3. Tenant wants premium template → 2 paths:
     a) SELF-PURCHASE: Tenant pays via Stripe → template_purchases created
        → auto-creates tenant_templates (status=active, source=purchase)
     b) SUPER ADMIN GRANTS: Super Admin goes to admin panel →
        selects tenant → selects template → assigns it manually
        → creates tenant_templates (status=active, source=manual)
  4. Super Admin can also BULK ASSIGN templates (e.g. all Pro plan tenants)
  5. Tenant admin picks default template per document_type in preferences

  Routes (superadmin.php):
    GET    /admin/templates                     → list all templates (catalog)
    POST   /admin/templates                     → create new template
    PUT    /admin/templates/{template}           → edit template (price, name…)
    DELETE /admin/templates/{template}           → soft-delete / deactivate
    GET    /admin/templates/{template}/tenants   → see which tenants have it
    POST   /admin/templates/{template}/assign    → assign to specific tenant(s)
    POST   /admin/templates/{template}/revoke    → revoke from tenant(s)
    POST   /admin/templates/bulk-assign          → assign to all tenants of a plan
    GET    /admin/tenants/{tenant}/templates     → see tenant's template access
    POST   /admin/tenants/{tenant}/templates/{template}/grant   → grant single
    POST   /admin/tenants/{tenant}/templates/{template}/revoke  → revoke single
    POST   /admin/tenants/{tenant}/templates/{template}/extend  → change expiry

  Controllers:
    App\Http\Controllers\SuperAdmin\TemplateCatalogController.php
    App\Http\Controllers\SuperAdmin\TenantTemplateController.php

  Services:
    TemplateAssignmentService::assign($tenantId, $templateId, $source, $expiresAt, $notes)
      → creates/updates tenant_templates row
      → sets status = active, activated_at = now(), activated_by = auth()->id()

    TemplateAssignmentService::revoke($tenantId, $templateId)
      → sets status = locked, deactivated_at = now()
      → if tenant had it as default → removes from preferences

    TemplateAssignmentService::bulkAssign($templateId, $planCode, $source)
      → finds all tenants on that plan
      → assigns template to each one

    TemplateAssignmentService::checkExpiry() [Cron daily]
      → finds tenant_templates where expires_at <= now() AND status = active
      → sets status = locked, deactivated_at = now()

5.1 Document Numbering (DocumentNumberService)

5.2 Calculators (QuoteCalculator / InvoiceCalculator)
─────────────────────────────────────────────────────
  Central recalculation to avoid bugs:

    line_subtotal = quantity × unit_price
    line_discount = apply discount_type/discount_value
    line_subtotal = line_subtotal - line_discount
    line_tax      = line_subtotal × tax_rate (or tax_group sum)
    line_total    = line_subtotal + line_tax

    Document totals:
      subtotal       = Σ line_subtotal
      discount_total = Σ line_discount
      tax_total      = Σ line_tax + charges_tax
      round_off      = rounding adjustment (if enabled)
      total          = subtotal - discount_total + tax_total + charges + round_off

    total_in_words (optional, via helper)

5.3 Snapshots (bill_from / bill_to / bank_details)

5.7 LoanService (Loans & Credit Management)
────────────────────────────────────────────

  LoanService — central service for all loan operations.

  ::createLoan($tenant, array $data): Loan
    → validates lender_type, principal, interest_rate, frequency
    → calculates total_amount based on interest_type (fixed/reducing)
    → sets remaining_balance = total_amount
    → sets status = active
    → calls generateInstallmentSchedule()

  ::generateInstallmentSchedule(Loan $loan): Collection
    → based on payment_frequency (monthly/quarterly/yearly)
    → for fixed interest:
        each installment.interest_amount = (principal × rate / 100) / nb_installments
        each installment.principal_amount = principal / nb_installments
    → for reducing interest:
        each installment.interest_amount = remaining_principal × (rate / 100 / periods_per_year)
        each installment.principal_amount = fixed principal portion
    → installment.total_amount = principal_amount + interest_amount
    → installment.remaining_amount = total_amount
    → installment.status = pending
    → bulk inserts all loan_installments

  ::applyPaymentToInstallment(LoanInstallment $inst, decimal $amount): void
    → inst.paid_amount += $amount
    → inst.remaining_amount = inst.total_amount - inst.paid_amount
    → if paid_amount >= total_amount → inst.status = paid, inst.paid_at = now()
    → elseif paid_amount > 0 → inst.status = partial
    → calls recalculateRemainingBalance(inst.loan)

  ::recalculateRemainingBalance(Loan $loan): void
    → loan.remaining_balance = Σ loan_installments.remaining_amount
    → if remaining_balance <= 0 → closeLoanWhenFullyPaid($loan)

  ::closeLoanWhenFullyPaid(Loan $loan): void
    → loan.status = closed
    → log activity: "Loan {reference} fully paid"

  Scheduled (Daily Cron):
  ::markOverdueInstallments(): void
    → loan_installments WHERE status IN (pending, partial)
      AND due_date < today
    → set status = overdue
    → optional: send notification to tenant

  LoanReportController (reports/loans):
    → total liabilities (Σ loans.total_amount WHERE status = active)
    → total remaining balance (Σ loans.remaining_balance WHERE status = active)
    → overdue installments count & total amount
    → upcoming payments (next 30 days)

  TemplateRenderService::render($documentType, $entity)
    1. Resolve default template code from tenant_template_preferences
    2. Load template_catalog by code
    3. Verify access via TemplateAccessService
    4. If denied → fallback to free default template
    5. Render Blade/HTML view with entity data
    6. Generate PDF via DomPDF/Browsershot
    7. Store generated PDF in documents table

      /**
       * All cases formatted for frontend <select> options.
       */
      public static function toSelectOptions(): array
      {
          return array_map(
              fn (self $s) => ['value' => $s->value, ...$s->details()],
              self::cases(),
          );

  <?php

  declare(strict_types=1);

  namespace App\Http\Controllers\Backoffice\Settings;

  use App\Enums\CompanyLegalStatus;
  use App\Http\Controllers\Controller;
  use App\Models\Tenancy\TenantSetting;
  use Illuminate\Http\Request;
  use Illuminate\Validation\Rule;
  use Inertia\Inertia;

  class CompanySettingsController extends Controller
  {
      // ─── SHOW ────────────────────────────────────────────
      public function show(Request $request)
      {
          $tenant  = $request->user()->tenant;
          $setting = TenantSetting::where('tenant_id', $tenant->id)->firstOrFail();

          return Inertia::render('Settings/CompanySettings', [
              'companySettings'    => $setting->company_settings ?? [],
              'legalStatusOptions' => CompanyLegalStatus::toSelectOptions(),
          ]);
      }

      // ─── UPDATE ──────────────────────────────────────────
      public function update(Request $request)
      {
          $tenant  = $request->user()->tenant;
          $setting = TenantSetting::where('tenant_id', $tenant->id)->firstOrFail();

          $validated = $request->validate([
              'name'         => ['required', 'string', 'max:255'],
              'legal_name'   => ['nullable', 'string', 'max:255'],
              'tax_id'       => ['nullable', 'string', 'max:50'],
              'email'        => ['nullable', 'email', 'max:255'],
              'phone'        => ['nullable', 'string', 'max:30'],
              'website'      => ['nullable', 'url', 'max:255'],
              'address'      => ['nullable', 'string', 'max:500'],
              'city'         => ['nullable', 'string', 'max:100'],
              'region'       => ['nullable', 'string', 'max:100'],
              'postal_code'  => ['nullable', 'string', 'max:20'],
              'country'      => ['nullable', 'string', 'max:5'],
              'legal_status' => ['required', Rule::enum(CompanyLegalStatus::class)],
          ]);

          // Build snapshot from the selected enum
          $legalEnum = CompanyLegalStatus::from($validated['legal_status']);

          $companySettings = array_merge(
              $setting->company_settings ?? [],
              $validated,
              [
                  'legal_status'                  => $legalEnum->value,
                  'legal_status_details_snapshot'  => $legalEnum->details(),
              ],
          );

          $setting->update(['company_settings' => $companySettings]);

          return back()->with('success', 'Paramètres de l\'entreprise mis à jour.');
      }
  }

  Notes:
    • TenantSetting::where('tenant_id', ...) ensures tenant isolation.
    • Rule::enum(CompanyLegalStatus::class) validates the value is one of
      sarl | sa | branch | subsidiary | sole_proprietorship.
    • The snapshot is written at save-time so historical documents retain
      the details even if the enum changes in a future release.

  /**
   * CompanySettings.jsx — Inertia page
   *
   * Props (from controller):
   *   companySettings    : object  (current JSON from tenant_settings)
   *   legalStatusOptions : array   (from CompanyLegalStatus::toSelectOptions())
   *
   * Behaviour:
   *   – <select> bound to legal_status
   *   – On change → display a details card underneath
   *   – On submit → Inertia.post to update route
   */

  import { useForm, usePage } from '@inertiajs/react';
  import { useState, useMemo } from 'react';
  import AuthenticatedLayout from '@/Layouts/AuthenticatedLayout';

  export default function CompanySettings() {
      const { companySettings, legalStatusOptions } = usePage().props;

      const form = useForm({
          name:         companySettings.name         ?? '',
          legal_name:   companySettings.legal_name   ?? '',
          tax_id:       companySettings.tax_id       ?? '',
          email:        companySettings.email        ?? '',
          phone:        companySettings.phone        ?? '',
          website:      companySettings.website      ?? '',
          address:      companySettings.address      ?? '',
          city:         companySettings.city         ?? '',
          region:       companySettings.region       ?? '',
          postal_code:  companySettings.postal_code  ?? '',
          country:      companySettings.country      ?? 'MA',
          legal_status: companySettings.legal_status ?? 'sarl',
      });

      // Find details for the currently selected status
      const selectedDetails = useMemo(
          () => legalStatusOptions.find(o => o.value === form.data.legal_status),
          [form.data.legal_status, legalStatusOptions],
      );

      function handleSubmit(e) {
          e.preventDefault();
          form.post(route('bo.settings.company.update'), {
              preserveScroll: true,
          });
      }

      return (
          <AuthenticatedLayout>
              <form onSubmit={handleSubmit} className="space-y-6 max-w-2xl">
                  {/* … other company fields (name, tax_id, address…) … */}

                  {/* ── Legal Status Select ──────────────── */}
                  <div>
                      <label className="block text-sm font-medium text-gray-700">
                          Forme juridique
                      </label>
                      <select
                          value={form.data.legal_status}
                          onChange={e => form.setData('legal_status', e.target.value)}
                          className="mt-1 block w-full rounded-md border-gray-300
                                     shadow-sm focus:border-indigo-500
                                     focus:ring-indigo-500 sm:text-sm"
                      >
                          {legalStatusOptions.map(opt => (
                              <option key={opt.value} value={opt.value}>
                                  {opt.label}
                              </option>
                          ))}
                      </select>
                      {form.errors.legal_status && (
                          <p className="mt-1 text-sm text-red-600">
                              {form.errors.legal_status}
                          </p>
                      )}
                  </div>

                  {/* ── Details Panel (French) ───────────── */}
                  {selectedDetails && (
                      <div className="rounded-lg border border-indigo-200
                                      bg-indigo-50 p-4 space-y-2">
                          <h3 className="font-semibold text-indigo-900">
                              {selectedDetails.label}
                          </h3>
                          <p className="text-sm text-gray-700">
                              {selectedDetails.description}
                          </p>
                          <ul className="text-sm text-gray-600 list-disc pl-5">
                              <li>
                                  <strong>Associés / Actionnaires :</strong>{' '}
                                  {selectedDetails.partners_or_shareholders}
                              </li>
                              {selectedDetails.min_capital_mad !== null && (
                                  <li>
                                      <strong>Capital minimum :</strong>{' '}
                                      {new Intl.NumberFormat('fr-MA', {
                                          style: 'currency',
                                          currency: 'MAD',
                                          maximumFractionDigits: 0,
                                      }).format(selectedDetails.min_capital_mad)}
                                  </li>
                              )}
                              {selectedDetails.notes && (
                                  <li>
                                      <strong>Remarques :</strong>{' '}
                                      {selectedDetails.notes}
                                  </li>
                              )}
                          </ul>
                      </div>
                  )}

                  <button
                      type="submit"
                      disabled={form.processing}
                      className="inline-flex items-center rounded-md bg-indigo-600
                                 px-4 py-2 text-sm font-medium text-white
                                 hover:bg-indigo-700 disabled:opacity-50"
                  >
                      {form.processing ? 'Enregistrement…' : 'Enregistrer'}
                  </button>
              </form>
          </AuthenticatedLayout>
      );
  }
